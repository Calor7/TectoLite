Welcome to TectoLite

This is the Beta of the restructured version I have been working on. I decided to release it directly as even though it is less tested, it has fewer bugs than the old version. If you still need to use the old version, feel free to visit TectoLite Legacy (available for a limited time). You can also grab GitHub versions to revert to a previous state or make personal changes.

This is the general manual for TectoLite (in progress). Hover over the flashing red highlights on the UI elements to learn more about each tool's function.

Getting Started

Start by drawing continental polygons using the toolbar on the left. Once you have a few plates, you can define their velocities and let the simulation run over time.

Keyboard Shortcuts

  • Space: Play/Pause simulation
  • Escape: Exit current tool / Close tutorial
  • Ctrl+Z / Ctrl+Y: Undo / Redo
  • V, H, D, E, S, L, G: Tool hotkeys (Select, Rotate, Draw, Edit, Split, Link, Fuse)

The Link Tool creates a parent/child hierarchical kinematic relationship between two plates.
The Fuse Tool merges two adjacent plates into a single continuous polygon.
