Welcome to TectoLite

This is the general manual for TectoLite, your Fantasy Plate Tectonics Simulator. Hover over the flashing red highlights on the UI elements to learn more about each tool's function.

Getting Started

Start by drawing continental plates using the toolbar on the left. Once you have a few plates, you can define their velocities and let the simulation run over time.

Keyboard Shortcuts

  • Space: Play/Pause simulation
  • Escape: Exit current tool / Close tutorial
